🛒 E-Commerce Web Application (ASP.NET MVC)

A full-stack e-commerce web application built using ASP.NET MVC and SQL Server.
The system simulates a real online store with complete product, user, and order management features, providing a smooth and interactive shopping experience.

🚀 Features

🔐 User Authentication & Authorization (Login / Register)
🛍️ Product Management (Add, Edit, Delete, View Products)
🛒 Shopping Cart System
📦 Order Processing & Checkout Flow
👤 Role-Based Access Control (Admin / User)
💾 Database Integration using SQL Server
🎨 Responsive User Interface using HTML, CSS, and JavaScript

🛠️ Technologies Used
ASP.NET MVC
C#
Entity Framework
SQL Server
HTML5
CSS3
JavaScript
Bootstrap
📂 Project Architecture

The project follows the MVC (Model-View-Controller) design pattern:

Models → Represent the data structure (Product, User, Order)
Views → Handle the user interface using Razor صفحات
Controllers → Manage application logic and user requests
